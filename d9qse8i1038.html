Objective 3: Implement a smart sensing station in a local watershed

To meat our objectives we will build a Smart Sensing Station that will host mussels and different sensors, such as toxicity sensors (to be developed in this project), and of the shelf sensors (i.e., water depth, temperature, and turbidity). A first prototype of the smart sensing station will be connected to a stormwater flume on the campus of Auburn University (Fig. \ref{894273}). Stormwater from this flume was found to be toxic to Daphnia pulex, if untreated \citep{Kuntz:2017}. The stormwater flow will be separated and one half will be treated with the Alabama bioretention mix and the other half will remain untreated. Both flows, treated and untreated, will then be routed in separate mussel containing chambers (Symbols A and B in Fig. \ref{894273}). Stream water will be directed to a third chamber (Symbol C in Fig. \ref{894273}) to monitor mussel behavior when exposed to stream water only and to capture eventual contaminant plumes that are in the stream already. During dry periods, stream water will be directed to all 3 chambers (A-C) to sustain the mussels.
Embedded computer system: We will use Raspberry Pis as frameworks for our sensor system (www.raspberrypi.org). Raspberry Pis are economical, yet powerful and affordable micro computer systems. Recently, Raspberry Pis have been used in geosciences \citep{moure_use_2015,morschhauser_low-power_2017}, automation \citep{noorshams_automating_2017,todd_fully_2017}, image analysis \citep{lopez-ruiz_portable_2017}, signal processing \citep{pasolini_raspberry_2017}, forecasting \citep{sukic_intra-minute_2017,richardson_low_2017}, as well as environmental and water quality monitoring .
Using a Raspberry Pi has many advantages over other computer or micro controller systems: They run a linux based operating system which makes it easy to implement the sensor algorithms (which were developed in objective 2) including specific R or python packages. Another advantage of the linux based operating system is that it is easy to implement a web server which means it would be possible to access the stations data directly through an internet browser. Raspberry Pis are equipped with basic I/O architecture like flash card reader, ethernet, USB, etc which lowers development costs. A powerful CPU and more RAM allow us to perform high performance computations directly on the smart sensing station.
