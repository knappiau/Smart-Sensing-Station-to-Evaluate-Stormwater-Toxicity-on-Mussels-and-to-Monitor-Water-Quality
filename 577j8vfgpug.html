Partial Least Square (Thorsten)

Partial least squares regression (PLSR) is a chemometric method similar to principal component analysis. PLSR can be used to analyze spectra measurements from hundreds of wavelengths and to reduce them to a smaller number of components. The number of components necessary is selected by minimizing the root mean square error of prediction (RMSEP). \citet{Etheridge_2014} used PLSR to determine nutrient, dissolved organic carbon, and total suspended solids concentrations in brackish water. A preliminary data set of PLSR applied to copper and zinc nitrate at different concentrations. A PLSR model was created from a training data set of 36 spectra using the pls package \citep{Mevik:2016} in R \citep{R:2017}. This model was used to predict the concentrations of copper, zinc, and nitrate concentrations of 10 validation spectra.
